# Web Firework Simulation

[Live Demo](https://miksin.github.io/web-firework-simulation/)

Canvas-based interactive firework simulation on browser.

## Features

- Canvas based
- Interactive
- Built on [Vue.js](https://vuejs.org/)
- Adjustable parameters
